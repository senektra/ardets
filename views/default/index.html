Development Life Cycle for Rapid Development

ARDeTS aspires to be a development life cycle technology that aims to help those working on a rapidly developing application. Originally created to support a project under rapid development, the idea has been taken further using web technology to create a more robust system. We are however still in early development and a lot of things are either broken or just not in place yet. We hope that early users will provide feedback via Git (we would really love it!).

Semantic Versioning Deployment

ARDeTS will use SEMVER versioning to automate some of the nitty gritty work developers should not need to worry about. This includes archiving old versions, maintaining production domain applications, and some other cool features. Some of them are implemented, but most of them are not. We hope to make it even easier in the feature with Git integration!
